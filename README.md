### Can ICO一套免费开源图标库

1. 将整个Can ICO图标库的目录复制到你的项目中
2. 在 `<head>` 标签中添加 `<link rel="stylesheet" href="canico.css">`
3. 在需要图标的地方添加对应 `class` 的 `<i>` 即可，已logo图标为例：`<i class="cd-candesign"></i>`



本地运行demo即可查看更多图例。
